# aa_sample_api

## Get Started

### install

```bash
git clone http://52.196.135.187/randd/aa_sample_api.git
cd aa_sample_api
yarn
# or
npm i
```

### start

```bash
yarn start
# or
npm start
# run `http://localhost:8888/`
```

```bash
PORT=9000 yarn start
# or
PORT=9000 npm start
# run `http://localhost:9000/`
```
